How to View, Change, Reset CMD Settings and Allow Script Execution

How to View CMD Settings

  1. View CMD Properties
    • Open CMD: Press Win + R, type cmd, and press Enter to open the Command Prompt.
    • Right-click on the CMD window title bar and select "Properties."
    • In the "Properties" dialog, switch to the "Font" tab to view the current font and font size.
  2. Check Registry Settings Using PowerShell
    • Open PowerShell: Press Win + X and select "Windows PowerShell" or "Windows PowerShell (Admin)."
    • Enter the following command to view CMD font settings: 
      Get-ItemProperty -Path "HKCU:\Console\"
  3. Check Registry Settings Using reg query Command
    • Open CMD: Press Win + R, type cmd, and press Enter to open the Command Prompt.
    • Enter the following commands to query CMD font settings: 
      reg query "HKEY_CURRENT_USER\Console" /v FaceName
reg query "HKEY_CURRENT_USER\Console" /v FontSize

How to Change CMD Settings

  1. Manually Change CMD Properties
    • Open CMD: Press Win + R, type cmd, and press Enter to open the Command Prompt.
    • Right-click on the CMD window title bar and select "Properties."
    • In the "Properties" dialog, switch to the "Font" tab, select the desired font and font size, then click "OK" to save changes.
  2. Change CMD Settings Using PowerShell Script
    • Open PowerShell: Press Win + X and select "Windows PowerShell" or "Windows PowerShell (Admin)."
    • Write a PowerShell script to modify CMD settings: 
      # Define font settings
$fontName = "Consolas"  # Change to your desired font name
$fontSize = 16          # Change to your desired font size

# Modify CMD settings in the registry
$regPath = "HKCU:\Console\"
Set-ItemProperty -Path $regPath -Name "FaceName" -Value $fontName
Set-ItemProperty -Path $regPath -Name "FontSize" -Value $fontSize

# Notify user to restart CMD
Write-Output "Please close and reopen CMD to apply changes."
    • Save the script as a .ps1 file and execute it.

How to Reset CMD Settings

  1. Reset CMD Properties
    • Open CMD: Press Win + R, type cmd, and press Enter to open the Command Prompt.
    • Right-click on the CMD window title bar and select "Properties."
    • In the "Properties" dialog, manually restore the settings to default values.
  2. Reset Registry Settings Using PowerShell
    • Use PowerShell to delete CMD registry settings: 
      Remove-Item -Path "HKCU:\Console\" -Recurse -Force
  3. Or, Reset Registry Settings Using CMD Command
    • Use CMD command to delete CMD registry settings: 
      reg delete "HKCU\Console" /f
    • Restart CMD to apply changes.

Allow PowerShell Script Execution

  1. Open PowerShell
    • Press Win + X and select "Windows PowerShell" or "Windows PowerShell (Admin)."
  2. Allow Script Execution
    • Enter the following command to allow script execution: 
      Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
    • The system will prompt for confirmation. Type Y and press Enter.
